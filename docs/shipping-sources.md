## 毎月の更新手順
1. 上記2ソースの最新値を確認（WebSearch で「ネコポス 料金 サイズ 2026」等、改定告知 info を確認）。
2. 変わっていたら **`engine.js`** の `SMALL`/`TAKKYU`/`YUPACK` を修正（コードが正）。
3. 連動して **`guide/hassou-houhou.html`**（②〜⑤の表）・関連記事（nekoposu / compact-vs / mercari-soryo-make）の数値も更新。
4. このファイルの表と「最終確認」日付、`engine.js` ヘッダの確認日を更新。
5. **`sw.js` の `CACHE` を1つ上げる**（engine.js/静的更新を配信するため）。
6. 反映確認（`soryo-tool.html`・`index.html` ③ で計算が変わるか）。
