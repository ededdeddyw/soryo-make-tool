# サービス戦略メモ（コンセプト・想定ユーザー・マネタイズ）

最終更新：2026-06-22

このメモは、処分ナビ／送料負け判定ツールの「何を・誰に・どう稼ぐか」を整理したもの。実装の判断に迷ったらここに立ち返る。

---

## 1. コンセプト
> **「これ、どうしよう？」を、“売る・譲る・捨てる”の出口まで迷わせない。家の不用品を、一番トクで一番ラクな手放し方へ案内するツール。**

一言で言えば **不用品の「出口ナビ」**。
競合（フリマアプリ／自治体ごみアプリ／買取サイト）がバラバラに存在する領域を、**「手放す意思決定」を軸に1か所に束ねている**のが強み。

- 入口：これどうする？（売る/譲る/捨てるの振り分け）
- 売るなら：送料負けせず売れるか（メルカリ送料計算）
- 捨てるなら：自治体ルール・収集日（ごみルール＋収集日リマインド）
- 譲る/買取/回収なら：適切な業者へ送客

## 2. 想定ユーザー
| 区分 | 誰 | 刺さる機能 | 主な流入経路 |
|---|---|---|---|
| **主（稼ぎ頭）** | 引っ越し・大掃除・断捨離でまとめて手放したい人（大型・急ぎ含む） | 処分ナビ→回収/買取/粗大ごみ | 「不用品 どうする」「○○区 粗大ごみ」 |
| 副A（集客・信頼） | フリマ出品で「送料負け／出す価値ある？」を知りたい初心者〜たまに出す層 | 送料負け計算 | 「メルカリ 送料 損」「ネコポス 厚さ」 |
| 副B（継続利用） | 新生活・一人暮らしでごみの出し方/収集日が分からない人 | ごみルール・収集日リマインド | 「○○区 ごみ 収集日」 |

**ファネルの肝**：副A・副Bの「便利な無料ツール」で検索流入＆信頼を作り、主層の高単価な手放し（回収・買取・引っ越し）で収益化する。
**主軸の置き方（推奨）**：「断捨離・引っ越しで手放したい人」を主軸に、「送料負け計算」を集客の尖りに使う＝収益と差別化の両取り。

## 3. マネタイズ（優先順位つき）
「集客＝便利さ・信頼」「収益＝出口への送客」の二段構え。お金は“捨てる/手放す”側（高単価リード）に厚い。

**◎ 高単価リード送客（本命）**
1. 不用品回収の見積もり比較 … 断捨離・大型・「運べない」層に自然。リード単価が高い。
2. 引っ越し一括見積もり … 断捨離ユーザーは引っ越し前後が多い。高単価。
3. 宅配/出張買取 … 「わからないけど高そう」「ブランド/家電」「運べない大型」導線に直結。

**○ 低単価・高頻度（相性◎で自然）**
4. 梱包資材アフィリ（Amazon/楽天）… 送料計算結果から「宅急便コンパクト専用箱・プチプチ・ダンボール」を提案。出品者が必ず買う＝CV高い。着手も軽い。
5. フリマアプリ新規登録 … 初心者層に「出す価値あり→メルカリ登録」。

**△ 将来（トラフィックが付いてから）**
6. アプリ内広告（控えめ）／プレミアム（広告非表示・複数地区のごみ通知・家族共有）

> 現状の手当て：外部リンクは index.html の `LINKS` 定数に集約済み＝ASP提携が取れたらURLを差し替えるだけ。判定ロジックも各出口（買取/回収/まとめ売り等）に対応済み。残るは各ASPの審査・提携。

## 4. 戦略の肝 ―― 「正直さ」が moat（差別化の堀）
直近で入れた「価値重視の判定」「送料負けを隠さない」「推測値を載せない」は、そのまま**信頼資産**。

- 買取アフィリ欲しさに、メルカリが得な人へ買取を勧める→**信頼が崩れてリピートが消える**。やらない。
- 正解は逆：**ユーザーにとってベストを正直に出し、その出口にだけ収益リンクを置く**。信頼が高いほど送客の質（成約率）も上がる。
- つまり **“フリマ側で信頼を作り、手放す側で稼ぐ”**。判定の中立性は絶対に売らない。

## 5. 次の一手
1. **ASP提携を取る（残タスクの本命）**：A8/もしも/Amazonアソシエイト＋不用品回収・引っ越し比較・宅配買取の各案件。
   - 取れたら差し替えるだけの状態は整備済み：梱包資材＝`AFFILIATE.amazonTag`（index/soryo）、買取/回収/譲る＝`LINKS`（index）。
2. ~~梱包資材リンクを送料計算の結果に追加~~ → **実装済み**（③結果に「🛒 必要な資材」リンク。発送方法に応じてAmazon検索）。②粗大判定にも買取/回収導線を追加済み。
3. SEO：robots.txt/sitemap.xml/JSON-LD/canonical/OGP は整備済み。次は**説明コンテンツ（送料・ごみ系）の拡充**と**対応自治体の追加**で検索流入を太らせる。
4. 入力の記憶（③）は実装済み。次は②の地区記憶や、①→③→②のクロス導線の磨き込み。
